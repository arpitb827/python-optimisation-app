Smart Coding with python and optimisation

There are various trips and tricks to develop the code for your application efficently with smart work.



1. Using of python built-in functions.

Here various built-in functions are available in python that makes your programming easy and optimised like lambda, filter, map, reduce .

lambda :

 it is anonymous function in python and also known as single line function in python.it is very usefull to perform operation on particular record set in one line.here example are described as below.

  Syntax: lambda arguments: expression

  Function: f1 = lambda x,y: x+y

  on calling: f1(1,2)

  Output will be: 3

filter :

 it is a function which can be used with lambda to filter the items from a given list based upon some expression condition.In below examle we are going to filter the even values from pass list [1, 2, 3, 4, 4, 5].

  Syntax: filter(lambda arguments: expression)

  Function: filter(lambda x: x%2==0 ,l1)

  on calling: filter(lambda x: x%2==0 ,l1=[1, 2, 3, 4, 4, 5])

  Output will be: [2,4,4]

map :

 it is a function which can be used with lambda to map the items of two list values into one list in single line or can be do any kind of mapping if we have two operand kind things.In below example we are going to sum the two list values and results the output.

  Syntax: map(lambda arguments: expression)

  Function: map(lambda x,y: x+y ,l1,l2)

  on calling: map(lambda x,y: x+y ,l1=[1, 2, 3, 4, 4],l2=[1,2,3,4,9])

  Output will be: [2, 4, 6, 8, 13]

reduce :

 it is a function which can be used with lambda to reduce the items of a list values or sum of the list values and results single value.like expample we are going to reduce the given list , [1,2,3,4]

  Syntax: reduce(lambda arguments: expression)

  Function: reduce(lambda x,y: x+y ,l1)

  on calling: reduce(lambda x,y: x+y ,l1=[1,2,3,4])

  Output will be: 10

2.Be careful to do while sorting, use generators.(By Key)

 Generators are very usefull when we talk about optimisation in python because they always stored the objects instead of whole value which is usefull for memory allocation.so wwhen we talk about sorting on large data set i prefer to use operator class of python, please follow the examples below.

  Input: l1= [(2,3,9),(1,5,6),(12,1,5)]

  step1: import operator

  step2: l1.sort(key=operator.itemgetter(1)) #here you cange it to 0,1 etc key index

  Output will be: l1= [(12, 1, 5), (2, 3, 9), (1, 5, 6)]

3.Try to play with set operations.

 Set operations are very usefull when we talk about optimise you can easily used on list sequence data structure.

  Input: l1= [(2,3,9),(1,5,6),(12,1,5)]

  step1: import operator

  step2: l1.sort(key=operator.itemgetter(1)) #here you cange it to 0,1 etc key index
